# ── BloodHound CE + collectors ────────────────────────────────────────────────
# BloodHound CE itself is the Postgres-backed web app — stand it up with SpecterOps'
# official `bloodhound-cli` (Go binary, curl the release / `go install`; NOT apt), which
# now owns the compose file under an XDG config dir. The COLLECTORS are what you install
# on the box:
bloodhound-ce-python # CE-compatible collector. NOTE: the 'bloodhound.py' apt pkg
#                    # is the LEGACY (≤4.3.1) collector — its zips don't ingest into CE
#                    # nxc's --bloodhound module also collects (see offensive.zsh `bhce`)
# rusthound-ce      → UPSTREAM (cargo install rusthound-ce): a single static Rust
#                     binary that emits CE-ingestible JSON — the fast ALTERNATE to
#                     bloodhound-ce-python on large domains (no Python chain to drop on
#                     a host). Not in Kali apt; not a redup fast-mover (collectors aren't
#                     daily-churn). Complement, not replacement.
# SharpHound (the C# collector) is dropped on the target, not installed here.
